## What is loseyourmoney.online?

loseyourmoney.online is a compact bilingual site that uses satire and financial education together in one static single-page app. It frames crypto, trading, and gambling-adjacent topics inside a guided story so the material can inform readers, spark reflection, and make risky market behavior easier to talk about.

The project targets the web and fits naturally on GitHub Pages. It mixes English and French, refers to public statistics, and stays deliberately lightweight through inline assets, hash-based routing, and a straightforward interface that does not depend on a heavy build pipeline.
